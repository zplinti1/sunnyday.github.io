原来每天读书时间不到一个小时

2018-11-12

读书之余，为了方便统计下读书时间，用了iHour这个APP。APP简单易用，我把最近需要读的书，分为三个项目：

  • 默克家庭医学手册
  • Java 36讲
  • 读库

每次开始读书，点击一下开始按钮，结束读书时，再点击一下结束，倒也方便。只是不曾想到，我每天看书的时间，平均下来，竟然每天不到一个小时。

我每天8点左右吃完晚饭，直到凌晨1-2点睡觉，中间差不多有5个小时的时间。而我，没有带小孩的烦恼，也不是经常性的加班，我把这些时间花哪去了？

令人深思！

决定从下周起，记录每天、每小时的时间使用情况，持续两周。两周后，再来总结一下。
